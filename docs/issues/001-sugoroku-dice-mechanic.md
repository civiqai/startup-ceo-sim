# Issue: 資金調達アクションに双六（サイコロ）演出を追加

## 概要
「資金調達」アクション選択時にサイコロを振る演出を入れ、出た目に応じて双六ボード上のマスを進み、止まったマスの効果が適用される仕組みにする。

## 背景
現在の資金調達は単純な乱数で結果が決まるだけで、プレイ体験として物足りない。サイコロ+双六の要素を入れることで、運と戦略のバランスが生まれ、ゲームとしての面白さが増す。

## サイコロアセット候補
- **Dice_Roll** (MIT) — https://godotengine.org/asset-library/asset/2865
  - 物理ベースのサイコロ演出、ボードゲーム用に設計
  - MITライセンスで自由に使える
- **Godot Dice Roller** (AGPLv3) — https://godotengine.org/asset-library/asset/3530
  - 高機能なUIコントロール、d6対応、値の事前指定可能
  - AGPLv3なのでライセンスに注意

## 仕様案

### 双六ボード（投資家ロード）
6〜10マスの円形ボード。マスごとに異なる効果：

| マス | 効果 |
|------|------|
| 大口投資家 | 資金+2000万 |
| エンジェル投資家 | 資金+500万、評判+15 |
| 条件付き出資 | 資金+1000万、評判-10 |
| VC面談成功 | 資金+1500万、次ターン連鎖イベント |
| 門前払い | 何も得られない |
| ライバルに先越された | 評判-20 |
| メディア露出 | ユーザー+300、評判+10 |
| アクセラレーター採択 | 資金+800万、プロダクト力+5 |

### フロー
1. 「資金調達」ボタンを押す
2. サイコロ振り演出（物理 or アニメーション）
3. 出た目の数だけボード上を進む
4. 止まったマスの効果を表示・適用
5. 通常のターン処理に戻る

### UI
- ボードは2Dのシンプルな円形/直線レイアウト
- 自分の位置をトークンで表示（ボード上の位置はターン跨ぎで保持）
- サイコロ演出はポップアップ or 画面遷移

## 拡張案
- 資金調達以外のアクションにも双六ボードを用意（採用ボード、開発ボード等）
- ボードのマスがゲーム進行で増える/変わる（フェーズ制と連動）
- 特定のマスに止まるとミニイベント発生
