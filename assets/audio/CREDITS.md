# Audio Credits

## Placeholder Audio Files

All audio files in this directory are **procedurally generated placeholder sounds**
created using Python's `wave` module. They are original works and not subject to
any third-party license restrictions.

### Files

| File | Description | Duration |
|------|-------------|----------|
| title_bgm.wav | Title screen BGM - upbeat arpeggio | ~8 sec |
| game_bgm.wav | Game screen BGM - calm pentatonic melody | ~10 sec |
| win_bgm.wav | Victory fanfare | ~4.5 sec |
| lose_bgm.wav | Game over somber music | ~4 sec |
| click.wav | Button click SFX | ~0.08 sec |
| notification.wav | Event notification SFX | ~0.3 sec |
| turn_advance.wav | Turn advance sweep SFX | ~0.25 sec |

## Recommended Replacements (Free/CC0 Sources)

For production-quality audio, consider replacing these placeholders with assets from:

- **Kenney.nl** (https://kenney.nl/assets?q=audio) - CC0, high quality game audio
  - UI SFX pack: https://kenney.nl/assets/interface-sounds
- **OpenGameArt.org** (https://opengameart.org/) - Various CC0/CC-BY licensed music
  - Search "corporate" or "business" for startup-themed BGM
- **Freesound.org** (https://freesound.org/) - Large CC0 sound library
- **Pixabay Music** (https://pixabay.com/music/) - Royalty-free music
